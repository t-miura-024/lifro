# タイマー音声ファイル

このフォルダにはタイマー機能で使用する音声ファイルを配置します。

## 仕組み

このフォルダ内の音声ファイル（`.mp3`, `.wav`, `.ogg`）は、アプリケーションが自動的に読み取り、タイマー設定画面の選択肢として表示します。

- 新しい音声ファイルを追加すると、自動的に選択肢に表示されます
- ファイル名（拡張子を除く）が選択肢の表示名になります
- DBにはファイル名（拡張子付き）が保存されます

## 推奨フォーマット

- **形式**: MP3（推奨）、WAV、OGG
- **長さ**: 0.1秒〜1秒程度
- **サンプルレート**: 44100Hz
- **ビットレート**: 128kbps以上

## 音声ファイルの追加方法

1. このフォルダ（`public/sounds/`）に音声ファイルを配置する
2. アプリケーションを再起動（または開発サーバーを再起動）する
3. タイマー設定画面で新しい音声が選択可能になる

## ファイル名について

- ファイル名は日本語でもOKです
- 拡張子を除いた部分が選択肢の表示名になります
  - 例: `カッ.mp3` → 選択肢には「カッ」と表示
- ファイル名は五十音順でソートされます
