# 画面遷移図
```mermaid
graph TD;
    LoginScreen[ログイン画面]
    LogManagementScreen[ログ管理画面]
    LogInputModal[ログ入力モーダル]
    LogDetailModal[ログ詳細モーダル]
    ExerciseManagementScreen[種目管理画面]
    StatisticsDashboardScreen[統計ダッシュボード画面]
    TimerManagementScreen[タイマー管理画面]
    TimerDetailModal[タイマー詳細モーダル]
    TimerOverlay[タイマー実行オーバーレイ]
    SettingsScreen[設定画面]

    LoginScreen -->|Googleログイン| LogManagementScreen
    LogManagementScreen <--> LogInputModal
    LogManagementScreen <--> LogDetailModal
    LogManagementScreen <--> ExerciseManagementScreen
    LogManagementScreen <--> StatisticsDashboardScreen
    LogManagementScreen <--> TimerManagementScreen
    LogManagementScreen --> SettingsScreen

    TimerManagementScreen <--> TimerDetailModal
    TimerManagementScreen --> TimerOverlay
    LogInputModal --> TimerOverlay
```
